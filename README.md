# hokkaido-zititai

北海道庁および道内全179市町村の連絡先・基本情報を一元管理するリポジトリです。

## 目的

- 各自治体との連携に必要な連絡先情報の管理
- 基本情報（人口・面積・自治体コード等）の参照
- 担当者・連携状況の記録

## 構成

```
北海道/
├── 北海道庁.md          ← 北海道庁・振興局の情報
├── 石狩地方/
│   ├── 札幌市/
│   │   └── 札幌市.md
│   ├── 江別市/
│   │   └── 江別市.md
│   └── ...
├── 空知地方/
├── 後志地方/
├── 胆振地方/
├── 日高地方/
├── 渡島地方/
├── 檜山地方/
├── 上川地方/
├── 留萌地方/
├── 宗谷地方/
├── オホーツク地方/
├── 十勝地方/
├── 釧路地方/
└── 根室地方/
```

## 各ファイルの記載内容

- **基本情報**: 自治体名、読み、自治体コード、地方、種別、人口、面積
- **連絡先**: 役所/役場所在地、代表電話番号、公式サイト、メール、FAX
- **担当者・連携情報**: 担当部署、担当者名、連絡先
- **メモ**: 自由記述欄
