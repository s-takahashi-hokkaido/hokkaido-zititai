# hokkaido-zititai

北海道の全自治体（179市町村）を地方別にディレクトリ構成で整理したリポジトリです。

## 構成

```
北海道/
├── 石狩地方/
│   ├── 札幌市/
│   │   └── 札幌市
│   ├── 江別市/
│   │   └── 江別市
│   └── ...
├── 空知地方/
├── 後志地方/
├── 胆振地方/
├── 日高地方/
├── 渡島地方/
├── 檜山地方/
├── 上川地方/
├── 留萌地方/
├── 宗谷地方/
├── オホーツク地方/
├── 十勝地方/
├── 釧路地方/
└── 根室地方/
```
